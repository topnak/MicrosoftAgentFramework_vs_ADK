# Copilot Instructions — MAF vs ADK Comparison Demo

## Project Overview

This is a **Streamlit multi-page application** that compares Microsoft Agent Framework (MAF) vs Google Agent Development Kit (ADK) side-by-side. It serves as a "Show and Tell" demo for enterprise conversational AI development.

## Rebuild Instructions

If this project needs to be rebuilt from scratch, follow these steps:

### Prerequisites

- Python 3.11+
- `pip install streamlit python-dotenv pandas azure-identity azure-ai-projects openai`
- (Optional) Azure AI Foundry project endpoint for live demo

### Key Design Patterns

1. **Side-by-side renderer** (`components/side_by_side.py`):
   - Takes MAF content (left, blue `#4A90D9`) and ADK content (right, green `#34A853`)
   - Uses `st.columns(2)` — stacks vertically on mobile via CSS `@media` queries
   - Every section ends with an `advantage_banner` callout

2. **Code snippets as Python constants** (`content/snippets_*.py`):
   - All code examples are string constants, not files
   - Rendered via `st.code()` for syntax highlighting
   - MAF snippets use OpenAI Responses protocol as baseline

3. **Live demo** (`live_demo/maf_chat.py`):
   - Connects to Azure AI Foundry via `AIProjectClient` + `DefaultAzureCredential`
   - Uses `openai_client.responses.create(stream=True)` for streaming
   - Falls back to mock mode if `.env` is not configured

4. **Responsive design** (`assets/style.css`):
   - Media queries at 1024px, 768px, 480px breakpoints
   - Columns stack vertically on mobile
   - Code blocks shrink font size on small screens
   - Framework labels and banners scale down

### Streamlit Page Naming Convention

Files in `pages/` must follow `N_PageName.py` format (number prefix for ordering).
Streamlit auto-discovers them for sidebar navigation.

### Content Update Rules

- **MAF snippets**: Update `content/snippets_maf.py` — use OpenAI Responses patterns
- **ADK snippets**: Update `content/snippets_adk.py` — use `google-adk` patterns
- **New sections**: Create new page in `pages/` with numeric prefix
- **Styling**: Modify `assets/style.css` — maintain blue/green accent convention

### Environment Variables

```
FOUNDRY_PROJECT_ENDPOINT=https://your-project.services.ai.azure.com
FOUNDRY_MODEL_DEPLOYMENT_NAME=gpt-4o

```
